LEssentials

Commands

GameMode

Changes the gamemode of an player.

Usage:

  • /gamemode <0,1,2,3,creative,survival,adventure,spectator> <name>
  • /gm <0,1,2,3,creative,survival,adventure,spectator> <name>
  • /gamemode <0,1,2,3,creative,survival,adventure,spectator>
  • /gm <0,1,2,3,creative,survival,adventure,spectator>

Fly

Changes the players fly state.

Usage:

  • /fly <name>
  • /fly

Teleport

Teleports the player to another player.

Usage:

  • /tp <name>

Vanish

Hides the player from every player without the required permissions.

Usage:

  • /vanish
  • /v

SpawnMob

Spawns a mob at the location the player is looking at.

Usage:

  • /spawnmob <mob>

EnderChest

Opens the enderchest of the player or another player

Usage:

  • /enderchest
  • /ec
  • /enderchest <name>
  • /ec <name>

Time

Sets the time to a pre set time.

Usage:

  • /day
  • /night

Weather

Sets the weather of the world to a pre set weather.

Usage:

  • /sun - sets the world to a Sunny day
  • /storm - sets the world to a Stormy day
  • /thunder - starts a thunder in the world

Message

Sends a private message to a player.

Usage:

  • /message <name> <message..>
  • /msg <name> <message..>

Sudo

Forces another player to execute a specific command.

Usage:

  • /sudo <name> <command>

Teleport All

Teleports every player to the executor.

Usage:

  • /teleportall

Heal

Heals a player.

Usage:

  • /heal
  • /heal <name>

Clear Chat

Clears the chat for every player.

Usage:

  • /clearchat <public,private>
  • /cc <public,private>

Give

Gives the player an item.

Usage:

  • /give <item>

Feed

Feeds an player.

Usage:

  • /feed
  • /feed <name>

Teleport Here

Teleports a player to the executor.

Usage:

  • /teleporthere <name>
  • /tphere <name>

Sign

Engrave your autograph onto an item with your personalized message.

Usage:

  • /sign <message..>

Rename

Rename an Item.

Usage:

  • /rename <name>

Warp

Warp yourself to an important place in the server.

Usage:

  • /warp <name>

Warps

See every important place in the server listed.

Usage:

  • /warps

Set Warp

Set an important place of the server.

Usage:

  • /setwarp <name>

Del Warp

Remove an important place of the server.

Usage:

  • /delwarp <name>

Home

Teleport yourself to one of your homes

Usage:

  • /home <name>

Set Home

Set one of your homes.

Usage:

  • /sethome <name>

Del Home

Delete one of your homes.

Usage:

  • /delhome <name>

Broadcast

Announce something to every player on the server.

Usage:

  • /Broadcast <message..>

Invsee

Inspect other players inventories.

Usage:

  • /invsee <name>

God

Make yourself invincible.

Usage:

  • /god
  • /god <name>

Trash

Open your personal trashbin to throw away unwanted items.

Usage:

  • /trash

Pay

Pay someone money.

Usage:

  • /pay <name> <amount>

Set Money

Set someones money to a specific amount.

Usage:

  • /setmoney <name> <amount>

Money

See your own or others current balance.

Usage:

  • /money
  • /balance
  • /bal
  • /money <name>
  • /balance <name>
  • /bal <name>

Sell

Sell the item your holding for money.

Usage:

  • /sell

Buy

Buy an item from the server.

Usage:

  • /buy <item>

Price

Get the current price of an item

Usage:

  • /price <item>
  • /price <item> <amount>

Wartung

Toggle the maintenance status of the server.

Usage:

  • /wartung

Event

Host or Join an Event.

Usage:

  • /event <join,create,delete>

TPA

Request a teleport to a player.

Usage:

  • /tpa <name>

TPAccept

Acceot a teleport request from a player.

Usage:

  • /tpaccept

TPACancel

Cancel a teleport request.

Usage:

  • /tpacancel

Reply

Reply to a private message from a player.

Usage:

  • /reply <message..>
  • /r <message..>

Undo Buy

Undo a unwanted purchase from the server.

Usage:

  • /undobuy

Skull

Get an head from a player.

Usage:

  • /skull <name>

Craft

Open your portable Crafting Table.

Usage:

  • /craft

Ban

Ban a player.

Usage:

  • /ban <name> <reason>
  • /ban <name> custom <reason>

UnBan

Unban a player.

Usage:

  • /unban <name>

Tempban

Temporarly ban a player.

Usage:

  • /tempban <name> <time>

Create Holo

Create a hologram.

Usage:

  • /createholo <text..>

Delete Holo

Delete a holo using its ID.

Usage:

  • /delholo <holoId>

Get Holo

Get the nearest holo id.

Usage:

  • /getholo

Guild

Use the LEssentials-Guild System.

Usage:

  • /guild create <name>
  • /guild <menu,leave,delete>
  • /guild invite <name>
  • /guild accept <name>

Gift

Gift a player the item your holding.

Usage:

  • /gift <name>

Open Gift

Open the gift you got from a player.

Usage:

  • /opengift

Usage:

  • /friends list
  • /friends <accept,decline,request> <name>
  • /f list
  • /f <accept,decline,request> <name>
